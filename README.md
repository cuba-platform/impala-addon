# CUBA Impala Add-on
## Overview

The add-on adds an ability to use Impala as additional datastore in the CUBA-based
applications.

## Compatibility with platform versions

| Add-on        | Platform      |

| 1.0-SNAPSHOT  | 6.10-SNAPSHOT |
